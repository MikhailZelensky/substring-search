## Описание

Бенчмарк для алгоритмов поика подстроки в строке.

Алгоритмы:
* Bruteforce
* Хеш-функция
* Конечный автомат
* Бойера Мура

## Использование
1.Запустите `benchmark.py`

2.Посмотрите отчёт в файле `report.png`, содержащий графики времени выполнения
тестов для различных алгоритмов.
